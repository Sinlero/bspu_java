Входной текстовый файл в первой строке содержит целое положительное число n,
в остальных строках файла располагается текст.
Число n задает некоторое количество текстовых строк в файле in.txt.
Написать программу, читающую n первых текстовых строк из файла in.txt в одномерный массив
строк с заданным размером n, и выводящую на консоль содержимое массива в обратном порядке.
PS. Для преобразования строки(String) str в целое чиcло можно использовать
следующую конструкцию:
int a = Integer.valueOf(str).intValue();
